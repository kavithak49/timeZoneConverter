TimeZone Extension/Addon
-------------------------

This extension lets a user convert time from one zone too another.

Steps to add the temporary extension to the firefox browser
-----------------------------------------------------------

1) Clone the repository into a local folder.
2) Open the firefox browser and type, about:debugging in the firefox address bar.
3)After the debugging page opens, click on This firefox option on the left panel.
4)Click on load temporary Addon
5) Browse to the location where you have cloned the timeZoneCoverter repository and choose the manifest.json file.
6) Now the addon icon is available in the firefox toolbar.
7) Click on the addon and a popup appears
8) Choose the time in 24 hours format
9) HH -> hours
10) MM -> minutes
